== Installation ==
1. Download *AntiVirus* plugin
1. Unzip the archive
1. Upload the folder *antivirus* into *../wp-content/plugins/*
1. Go to tab *Plugins*
1. Activate *AntiVirus*
1. Edit settings
1. Ready
